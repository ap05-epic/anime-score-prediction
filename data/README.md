# Data

The dataset is not committed to git (see `.gitignore`). To run the notebooks:

1. Download `anime-dataset-2023.csv` from the Kaggle dataset [dbdmobile/myanimelist-dataset](https://www.kaggle.com/datasets/dbdmobile/myanimelist-dataset).
2. Place it in this folder so the path is `data/anime-dataset-2023.csv`.

Only `anime-dataset-2023.csv` is used. The `users-details-2023.csv` and `users-score-2023.csv` files are not needed for this project.
